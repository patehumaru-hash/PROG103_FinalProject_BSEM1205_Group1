# AgriTrack SL

A GUI-based farmer record management system built with Python (tkinter + SQLite),
developed for PROG103 - Principles of Structured Programming.

## Overview
AgriTrack SL tracks smallholder farmers in Sierra Leone, supporting SDG 2 (Zero Hunger).

## Features
- Secure login
- Farmer records dashboard (25+ seeded records)
- Search and filter (gender, status, date)
- Bar, pie, and line chart visualizations

## How to Run
1. `pip install matplotlib`
2. `python main.py`
3. Login: admin / admin123
